#config  #vlan

1. Enter global configuration 
```vim
configure terminal
```
2. Enter vlan
```vim
int vlan#
```
3. enter ip 
```vim 
ip address 0000.0000.0000.0000 255.255.255.0
```
4. no shutdown 
```vim 
no shutdown 
```
5. exit 
```vim 
exit 
```
